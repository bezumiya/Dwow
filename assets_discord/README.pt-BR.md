# Assets para o Discord Developer Portal

**[English](README.md)**

Esta pasta contém as imagens prontas para upload usadas pelo Rich Presence do
Dwow. Abra sua aplicação no [Discord Developer Portal](https://discord.com/developers/applications)
e envie cada PNG em **Rich Presence → Art Assets**. O nome do arquivo sem
`.png` deve ser mantido como a chave do asset.

| Arquivo | Chave no Discord |
|---|---|
| `wow_classic.png` | `wow_classic` |
| `class_warrior.png` | `class_warrior` |
| `class_mage.png` | `class_mage` |
| `race_orc_male.png` | `race_orc_male` |
| `race_nightelf_female.png` | `race_nightelf_female` |

Não altere maiúsculas, espaços ou o formato das chaves. O companion gera esses
nomes automaticamente usando classe, raça e gênero do personagem.

## Conteúdo

- `wow_classic.png`: imagem padrão do Rich Presence.
- `class_*.png`: ícones das 11 classes suportadas.
- `race_*_male.png` e `race_*_female.png`: retratos das 13 raças suportadas.

O Discord pode levar alguns minutos para disponibilizar novos assets. Reinicie
o companion caso uma imagem recém-enviada não apareça.
